


CyberMitra 🛡️🤖
Empowering Digital Literacy and Safety through AI-Driven Interaction, Real-Time Scam Simulation, and Gamified Education.

CyberMitra is an interactive, all-in-one digital safety ecosystem designed to bridge the gap between complex cybersecurity concepts and everyday internet users. By combining real-time AI assistance, immersive sandboxed simulations, and gamified learning, CyberMitra transforms passive awareness into active digital defense.

🚀 Core Features
AI-Driven Real-Time Mascot: An intelligent, empathetic digital assistant powered by advanced LLMs that provides instant, contextual guidance on privacy settings, suspicious links, and immediate incident response steps.

Scam Simulation Engine: A safe, sandboxed environment mimicking real-world threats (phishing emails, smishing texts, and social engineering loops). Users can interact with live threats risk-free to build muscle memory against social engineering.

Deepfake Recognition Lab: An educational module breaking down the anatomy of synthetic media, teaching users how to spot visual and auditory anomalies in AI-generated content.

Gamified Learning Ecosystem: A quiz-based learning arena inspired by Kaun Banega Crorepati (KBC), using dynamic scoring, interactive lifelines, and progressive difficulty tiers to reinforce cybersecurity hygiene.

🏗️ Technical Architecture & Stack
CyberMitra is architected for speed, modularity, and seamless state management.

Frontend: React 18 + TypeScript + Vite (for lightning-fast HMR and type-safe component development)

Styling: Tailwind CSS (for responsive, utility-first UI/UX design)

AI Orchestration: Google GenAI SDK / Gemini API (driving the intelligent response loop for the mascot)

State Management & Routing: React Context API / React Router DOM

📈 Impact & Intended Outcomes
Behavioral Change: Shifts user behavior from reactive panic to proactive validation when dealing with digital anomalies.

Democratizing Security: Translates complex technical jargon (e.g., phishing, deepfakes, social engineering) into accessible, relatable concepts for all age demographics.

Zero-Risk Exposure: Allows non-technical users to experience a cyberattack firsthand in a sandboxed environment, reducing real-world vulnerability rates.
## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
